> 단항 연산자(unary operator) : 피연산자 1개. 예) a++
>
> 2항 연산자(binary operator) : 피연산자 2개. 예) a < b
>
> 3항 연산자(ternary operator) : 피연산자 3개. 예) a ? b : c
>
