## 검색의 종류
a. 배열 검색
b. 선형 리스트 검색
c. 이진 검색트리 검색 


### a.배열 검색
1. 선형 검색: 무작위로 늘어놓은 데이터 모임에서 검색을 수행합니다.
2. 이진 검색: 일정한 규칙으로 늘어놓은 데이터 모임에서 아주 빠른 검색을 수행합니다.
3. 해시법: 추가, 삭제가 자주 일어나는 데이터 모임에서 아주 빠른 검색을 수행합니다.
    - 체인법: 같은 해시 값의 데이터를 선형 리스트로 연결하는 방법
    - 오픈 주소법: 데이터를 위한 해시 값이 충돌할 때 재해시하는 방법
