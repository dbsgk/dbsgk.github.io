## 136. Single Number


```java


  class Solution {
      public int singleNumber(int[] nums) {
          int test;
          int count;
          int single = 0;
          iFor: for(int i = 0; i < nums.length; i++) {
              test = nums[i];
              count = 0;
              for(int j = 0; j < nums.length; j++) {
                  if(test == nums[j]) {
                      count++;
                  }

              }
              if(count == 1) {
                  single = test;
                  return single;
              }

          }
          return single;
      }
  }


```
